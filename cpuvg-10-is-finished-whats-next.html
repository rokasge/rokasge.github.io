## CPUVG 1.0 is finished. What's next?

I have finished first version of the project I was working on. I am quite satisfied with myself because it was a big project for me and I managed to code every single thing I have planned for version 1.0 and more. I didn't manage to meet the deadline I was aiming for but 4 days past the deadline is not that much especially when it was a personal project and I was learning things. During the time I was working on CPUVG1.0, I came up with new ideas for what I can implement for the next version if I decide to work on it. Some ideas for CPUVG 2.0:

  • Move my script to Google Drive to make it easier for the main script and Google Apps Script to work together.
  • Google Apps script using the same data files as the main script to generate descriptions of the videos.
  • Thumbnail maker inside the main script, thumbnails to be automatically set by Google Apps Script.
  • Review and make a better structure of data files.
  • Data scraper for different sites.
  • Make a python script to generate simple web pages for displaying data - "CPUVG"/item/item-name.
  • Review design and make changes(it can always look better): fonts, backgrounds, sections, pictures.

While these are a lot already I just leave these for when I get back to the project, I might think of new ideas while doing other things. For now, I am going to take a break of around 2-3 months from this project because I have other stuff to do.
